**CSE 101 Programming Assignment 5**

In this assignment, we built an Integer List ADT in C++, and used it to perform a perfect shuffle on an input integer from the user. The following files are required for this project:

1. List.h: 
	- The header file for the List ADT's functions and procedures
	- Declares access functions, constructors-deconstructors, manipulation procedures, and other operations
2. List.cpp: 
	- The corresponding c++ file for List.h, containing the functions and procedures declared in the header file
	- Uses 2 Node pointers
3. ListTest.cpp: 
	- A test client for the List ADT intended to test the functionality of all ADT operations
4. Shuffle.cpp: 
	- The main program that returns two columns displaying a perfect shuffle of the input integer
	- Implements the List ADT
	- Takes one command line argument 
5. Makefile: 
	- The Makefile creates an executable binary called Shuffle, as well as ListTest and ListClient
	- Also includes a clean target that removes any associated executables/.o files
