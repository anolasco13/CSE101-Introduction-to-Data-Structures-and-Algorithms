# CSE 101 – Introduction to Data Structures and Algorithms
**University of California, Santa Cruz**

Introduction to abstract data types and the basics of algorithms. Topics include linked lists, stacks, queues, hash tables, trees, heaps, and graphs, as well as Big-O analysis of simple algorithms. All programming assignments are implemented in C/C++.

---

## Repository Structure

Each programming assignment lives in its own directory with a dedicated README describing the problem, approach, and build instructions.

| Assignment | Topic |
|------------|-------|
| `pa1/` | Integer List ADT |
| `pa2/` | Graph ADT |
| `pa3/` | Graph DFS Algorithm |
| `pa4/` | Matrix Calculator |
| `pa5/` | Integer List ADT (C++) |
| `pa6/` | BitInteger ADT |
| `pa7/` | Dictionary ADT |
| `pa8/` | Dictionary Red-Black Tree ADT |

---

## Building & Running

Each assignment can be compiled using the provided `Makefile`:

```bash
cd pa1
make
./pa1
```

To clean build artifacts:

```bash
make clean
```

---

## Topics Covered

- Abstract Data Types (ADTs)
- Linked Lists (singly, doubly)
- Stacks & Queues
- Hash Tables
- Trees (binary search trees, AVL)
- Heaps & Priority Queues
- Graphs (BFS, DFS, shortest paths)
- Big-O / Asymptotic Analysis
